# longhandarchive.github.io

The development site for [The Longhand Archive](https://github.com/longhandarchive) — documenting how a solo archival system gets built in public.

Built with Jekyll and served via GitHub Pages at [longhandarchive.github.io](https://longhandarchive.github.io).

## Structure

- `_pages/` — standing reference pages (approach, architecture, decisions)
- `_posts/` — authored narrative pieces, published when there is something to say
- `assets/` — images and diagrams
- `_config.yml` — Jekyll configuration
